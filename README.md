# 23met Helper

Мобильная веб-форма для iPhone и серверный сборщик цен с 23met.ru через Chromium/Playwright.

## Локальный запуск

```bash
npm install
npx playwright install chromium
npm start
```

Откройте http://localhost:3000.

## Развёртывание на Render

1. Создайте пустой репозиторий GitHub и загрузите файлы проекта.
2. В Render выберите New → Blueprint и подключите репозиторий.
3. После запуска откройте выданный адрес на iPhone.
4. Safari → Поделиться → На экран «Домой».

## Важное ограничение

23met может ограничивать серверные IP ошибкой 429. Если это произойдёт и в Chromium, потребуется прокси с российским резидентским IP либо запуск сервера на домашнем/офисном компьютере. Селекторы страницы могут потребовать корректировки после первого живого теста.
